The Basic Language of The Web: HTML

posted by salma haji on monday, june 21st 2027

All modern websites and web applications are built using
three fundemental technologies: HTML, CSS and javaScript.
We will learn what HTML is all about, and why
you too should learn it.

What is HTML?

HTML, or HyperText Markup Language, is the standard language used to create
and structure content on the web. It consists of a series of elements,
represented by tags, that define the structure of a webpage.
HTML is used to specify headings, paragraphs, links, images, lists,
and other types of content.

in HTML, each element is made up of 3 parts
1. The opening tag
2. The closing tag
3. The actual element

You can learn more at MDN Web Docs


Why should you learn HTML?

there are countless reasons for learning the fundamental language of the web.
Here are 5 of them

  • to be able to use the fundamantal web dev language
  • To hand-craft beautifull websites instead of relying on tools like worpress or wix
  • To build web applications
  • To impress friends
  • To have fun
